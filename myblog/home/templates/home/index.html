I made this website for fun.

It wouldn't have been possible without the following open source technologies:





Python is an open-source programming language. The syntax is super-readable and code that's "pythonic" actually seems to take on a certain beauty. If you're thinking about learning programming, I think python is a great first language.

Django is an open-source web framework for Python. Django provides a ton of amazing tools for interacting with databases, writing logic into your website, and dynamically rendering HTML to send to a browser. I've only scratched the surface so far, but I look forward to learning more as I add features to the site.

Pycharm is a fantastic IDE for Python. The folks at JetBrains make the Community Edition available for free. Pycharm makes it easy to write, style, and refactor code so it's readable and consistent.

HTML (HyperText Markup Language) is the language browsers interpret to organize what you see. HTML5 offers a bunch of new features making it easier to read, write, and organize the language that makes the internet possible.

JavaScript is a programming language that runs inside the user's web browsers. It allows for elements to be dynamically edited on the fly. I have barely used any of it, but what I have used adds a nice touch and makes the site a little more responsive. I'm definitely going to try and incorporate JavaScript more as the site grows.

CSS (Cascading StyleSheets) is the language used to tell browsers how things delivered via HTML should look. Everything from font size to image locations and element re-shaping based on window size happens in CSS. This is a deep rabbit hole, and I'm just getting started.

Fedora is a popular distribution of the Linux operating system. I chose Linux because of the freedom it gives me to customize my environment, and I chose Fedora in particular sort of at random. I haven't been disappointed.

GitHub is THE place to create remote code repositories using Git. Keeping a remote backup of the code that runs any hobby project I've been working on has allowed me to tinker even when I'm not on my machine. It's also amazing to see what other people are up to in the open-source world. There are some truly amazing projects out there.

Bootstrap is a free HTML/CSS/Javascript framework that helps design websites that are responsive and good looking. By using a mobile-first focus, Bootstrap makes it easy to ensure your website looks exactly how you want on even the smallest screens.
